# Anti-slop rules, with the fix for each tell

Read this before you draft, not after. Writing plainly the first time is
faster than writing inflated and rewriting it, and it is the actual goal here,
not passing a check. The script below catches what slips through; these are
the judgment calls a script cannot make. The full sourced version lives at
https://selwynuy.dev/d/writing-without-ai-slop.md.

## Mechanical (the script flags these)

| Tell | Fix |
| --- | --- |
| Em-dash | Use a period or a comma. Rewrite the sentence if it needs the pause. |
| "not just X, but Y" | State Y plainly. Drop the contrast. |
| "it's worth noting that" | Delete it. Say the thing. |
| "in today's world / fast-paced" | Delete the opener. Start at the point. |
| delve, leverage, utilize, seamless, robust, elevate, unleash, harness | Plain verbs: dig into, use, use, smooth, solid, raise, release, use. |
| résumé, café, naïve, fiancé(e), cliché | Plain English spelling: resume, cafe, naive, fiance/fiancee, cliche. |

## Judgment calls (read for these yourself)

- **Manufactured contrast.** Any "it's not about A, it's about B" framing that
  exists to sound profound. Cut to B.
- **Hedging.** "somewhat", "arguably", "in many ways", "can help to". If it is
  true, say it. If it is not, cut it.
- **Empty intensifiers.** "truly", "really", "very", "incredibly". They add
  nothing. Delete.
- **Listing three adjectives.** "fast, reliable, and scalable." Pick the one
  that matters or show it instead of claiming it.
- **Throat-clearing conclusions.** "In conclusion", "At the end of the day",
  "Ultimately". Just write the last sentence.
- **Vague authority.** "studies show", "experts agree" with no source. Name it
  or drop it.

## The test

Read it aloud. If it sounds like a person who knows the subject talking to
another person, it passes. If it sounds like a press release, rewrite.
